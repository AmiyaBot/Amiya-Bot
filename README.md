# Amiya-bot

基于 [mirai-console](https://github.com/mamoe/mirai-console)
和 [mirai-api-http](https://github.com/project-mirai/mirai-api-http) 的 QQ 聊天机器人<br>
项目名称源于游戏 [《明日方舟》](https://ak.hypergryph.com/) 的女主角"阿米娅"，其主题与核心功能也与其相关。<br>
